# vuepress-plugin-live2d-cat

## How to use

1. install `npm i vuepress-plugin-live2d-cat -S`

2. add in `config.js`

```js
module.exports = {
  plugins: [
    "vuepress-plugin-cat",
  ]
}
```

## Tip

- This plugin supports `vuepress 1.x`
- This plugin will randomly change the white cat or black cat when router path change.
